# Netflix Data Analysis

## Overview
This project analyzes the Netflix dataset to uncover insights about movies and TV shows, including trends over time, content distribution, and popular categories.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
- **Name:** Netflix Titles Dataset  
- **Source:** Kaggle  
- **File:** data/netflix_titles.csv  
- **Description:** Contains details about Netflix content such as title, director, cast, country, release year, rating, and duration.

## Key Insights
- Most content was added after 2015  
- Movies are more common than TV shows  
- USA produces the highest number of titles  

## Project Files
- `netflix_data_analysis.ipynb` → Main analysis notebook  
- `data/netflix_titles.csv` → Dataset  

## Conclusion
This analysis helps understand how Netflix content has evolved over time and highlights key patterns in the dataset.
